    • $ CSS Guide



    Cascading Style Sheets (CSS) is a fundamental technology used to define the presentation and layout of web pages. It works alongside HTML to structure and design the visual elements of a website. This guide will cover CSS from basic syntax to more advanced techniques.


    $ Introduction to CSS



    CSS stands for Cascading Style Sheets. It is a language used to describe the presentation of a document written in HTML or XML. CSS controls the style, layout, and design of multiple web pages at once, making it a powerful tool for web development.


    $ Basic CSS Syntax



    CSS uses a simple syntax to apply styles to HTML elements. Here's an example: 


    
 /* Selects all "p" elements and sets font size to 16px */
 
    
p {
font-size: 16px;
color: #333;
}


    $ Selectors



    Selectors are patterns used to select HTML elements to style. Common selectors include:


  • Element Selector: Targets specific HTML elements (p, h1, div)
  • Class Selector: Targets elements with a specific class (.btn, .container)
  • ID Selector: Targets a unique element with a specific ID (#header, #sidebar)
  • Attribute Selector: Targets elements based on attributes ([type="text"])

    • $ CSS Properties



    CSS properties define how HTML elements are displayed. Some common properties include:


  • color: Sets the text color
  • font-size: Sets the font size
  • margin, padding: Controls spacing around elements
  • border: Defines the border around elements
  • background: Sets background properties like color or image.

    • $ Layout and Positioning



    CSS offers various methods to control layout and positioning:


  • Floats: Positions elements to the left or right within their container.
  • Flexbox: Provides a more efficient way to layout items in a container.
  • Grid: Allows for complex layouts with rows and columns.

    • $ Responsive Design



    Responsive design ensures your website looks good on all devices. CSS techniques for responsiveness include:


  • > Media Queries: Apply styles based on screen size (@media (max-width: 768px))
  • > Viewport Meta Tag: Sets the viewport width for mobile devices (>meta name="viewport" content="width=device-width, initial-scale=1.0"<)


    • $ CSS Frameworks



    Frameworks like Bootstrap and Foundation offer pre-designed CSS components and layouts to speed up development.


    $ CSS Preprocessors



    CSS preprocessors like Sass and Less extend CSS with variables, functions, and nested rules, making CSS more maintainable and efficient.
